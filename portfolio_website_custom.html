Chatbot System for Universities

Our project is a Chatbot system, That is created to address all the queries for all types of users such as students, employees, faculty, parents and others. The queries can be related to Admissions, Course Registration, On-Campus events, Employment Opportunities and many more. This chatbot can be integrated into the Universities Existing System, and can have features like Scheduling Appointments, Checking Application Status etc. Using Natural Language Processing and Machine Learning Algorithms, this chatbot will understand the user inputs, and will be responding accordingly. This will be integrated to a Database to provide updated information. The final goal of this chatbot is to enhance user experience, save time, reduce Load and Burden to the IT Department, and increase operational efficiency.
